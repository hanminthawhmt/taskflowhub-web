# Rules

Before creating a component

Search this library.

Reuse whenever possible.

Only create new components when no existing component satisfies the requirement.

Shared components belong inside

components/

Feature-specific components belong inside

features/<feature>/components

Never duplicate components with similar responsibilities.

Favor composition over inheritance.
