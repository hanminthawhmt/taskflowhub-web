# Task Management SaaS Frontend

## Project Overview

This project is a React + TypeScript frontend for a Task Management SaaS platform.

## Source of Truth

Before starting any task, read every document inside `/docs`.

The documents include:

- backend-api.md
- frontend-tech.md
- ui-guidelines.md
- architecture.md
- coding-standards.md
- workflow.md
- api-conventions.md
- component-library.md

## Rules

- Never invent backend APIs.
- Follow backend-api.md exactly.
- Follow the architecture document.
- Use reusable components.
- Use React + TypeScript.
- Use Tailwind CSS and shadcn/ui.
- Use TanStack Query for server state.
- Use Zustand for client state.
- If an API is missing, document it in `new-endpoint-suggestions.md`.
